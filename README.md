# 💳 Concurrent Bank System (Java)

A multi-threaded Java application that demonstrates core concurrency concepts such as threads, synchronization, and locking mechanisms using a real-world bank account simulation.

---

## 🚀 Features

- Multi-threaded transaction handling
- Thread-safe operations using ReentrantLock
- Simultaneous deposit and withdrawal operations
- Race condition prevention
- Real-time console output of transactions

---

## 🧠 Concepts Covered

- Java Threads
- Concurrency
- Synchronization
- Locks (ReentrantLock)
- Critical Section Handling
- Thread Lifecycle (start, join)
